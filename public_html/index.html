I started with my c++/python coordinates library as a foundation for implementing Astronomical Algorithms by Jean Meeus. This uses flask and gunicorn to create a http interface. Since I am requesting global position data from the browser, I added nginx as a reverse proxy and to act as a TLS endpoint for my letsencrypt certs. I also modified the docker.io nginx container to run cron to renew the certs and rotate the logs. Lately, I have added mongodb in another container along with another flask/gunicorn UI to record observations. This also implements account logins.
